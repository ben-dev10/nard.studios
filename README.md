# nard.studios

This repo contains the source code for the 3rd revision of my personal website (and a web design studio in the works).

![nard,studios og](./public/og.webp)

## Installation

```bash
npm install
fumadocs-mdx && next dev --turbopack
```

Packages and technologies used:

- Next.js, TailwindCSS
- Fumadocs (for mdx processing)
- Vercel analytics
